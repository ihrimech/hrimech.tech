Sorting

There will come a time when you will need to list and sort resources. Hyde allows you to walk the site tree and sort the resources by the predefined settings in your configuration file.

You can list and sort by name all your content files.

  • Contact (contact.html)
  • Grouper (grouper.html)
  • Index (index.html)
  • Overview (overview.html)
  • Profilage visual vm (profilage-visual-vm.html)
  • Scala training (scala-training.html)
  • Sorter (sorter.html)
  • Spark training (spark-training.html)
  • Tagger (tagger.html)

Or list only those in the current node (folder). In this case that would be all advanced topics.

  1. Overview
  2. Sorter
  3. Grouper
  4. Tagger
